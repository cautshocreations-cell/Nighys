# Les Night - Site d'Officialisation

Site web pour la demande d'officialisation du gang "Les Night" avec le thème cyan/bleu clair.

## Structure du Site

### Sections Incluses:

1. **Trame du Groupe** - Histoire, philosophie, valeurs et objectifs du gang
2. **Zone Revendiquée** - Territoire et zones contrôlées
3. **Actions Réalisées** - Tableau chronologique des actions du gang
4. **Membres** - Tableau des membres avec grades et couleurs distinctives
5. **Galerie Photos** - Collection de photos du gang

## Comment Utiliser

1. Ouvrez le fichier `index.html` dans votre navigateur
2. Modifiez le contenu selon vos besoins:
   - Remplacez les textes de la trame avec votre histoire
   - Ajoutez vos zones revendiquées
   - Complétez le tableau des actions avec vos dates et détails
   - Renseignez les identités RP et Discord de vos membres
3. Ajoutez vos photos dans le dossier `photos/` et mettez à jour les chemins dans le HTML

## Hiérarchie des Grades

Les grades sont colorés de manière distinctive:
- **Chef** - Or (Gold)
- **Bras Droit** - Rouge-orangé
- **Lieutenant** - Violet
- **Membre** - Cyan (couleur du gang)
- **Recrue** - Vert clair

## Personnalisation

Pour modifier les couleurs ou le style, éditez le fichier `style.css`.
Les variables CSS sont définies au début du fichier pour faciliter les modifications.

## Navigation

Le site comprend une navigation sticky qui reste visible lors du défilement pour un accès rapide à toutes les sections.
